# Software_Gestao
Projeto desenvolvido visando criar um sistema que integre gestão acadêmica, RH e financeiro que possua granularidade suficiente para abastecer uma IA que ofereça analíticos a partir dos dados coletados do sistema. 

Project developed with the aiming of creating a system that connects academic, human resources, and financial administration that has sufficient granularity to feed an AI provides analytics using the data collected by the system
